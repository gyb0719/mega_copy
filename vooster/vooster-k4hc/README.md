# Vooster K4HC

Vooster project initialized with code K4HC.

## Quick Start

```bash
cd vooster-k4hc
npm install
npm start
```

## Project Structure

- `src/` - Source code
- `config/` - Configuration files
- `tests/` - Test files
- `docs/` - Documentation

## Scripts

- `npm start` - Run the application
- `npm run dev` - Run in development mode with watch
- `npm test` - Run tests
- `npm run lint` - Lint code
